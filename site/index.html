BoardClip

Find old clips, pin reusable snippets, label messy work, paste numpad macros, and sync through folders you already use.

Install

Recommended command

curl -fsSL https://boardclip.sh/install.sh | sh

Installs from the open-source repo into your user app folder. If BoardClip is already installed, this runs the standard update flow instead.

Built for daily repetition

Instant search

Open at your cursor, type, press Enter, and paste without turning clipboard history into admin work.

Numpad quick-paste

Assign your most-used snippets to 1-9 and paste them from anywhere.

Labels and pins

Keep durable snippets, tag work-in-progress, and filter a large clipboard fast.

Native cloud folders

Sync through Google Drive, OneDrive, and iCloud where available, with local files you can inspect.

Update

Update

Recommended command

curl -fsSL https://boardclip.sh/update.sh | sh

Pulls the latest source, refreshes dependencies, and relaunches the tray app.

Open source. Local-first. No hosted clipboard database. github.com/tobq/boardclip
