BoardClip

Clipboard history, snippets, and macros across devices.

Search old clips, pin reusable snippets, label work in progress, save image clips, assign numpad macros, and sync through cloud storage you already use.

Install

Recommended command

curl -fsSL https://boardclip.sh/install.sh | sh

Installs from the open-source repo. If BoardClip already exists, this runs the normal update flow.

Built for busy clipboards

Instant search

Open at your cursor, type, press Enter, and paste without turning clipboard history into admin work.

Numpad quick-paste

Assign your most-used snippets to 1-9 and paste them from anywhere.

Labels and pins

Keep durable snippets, tag work-in-progress, and filter a large clipboard fast.

Native cloud folders

Sync through Google Drive, OneDrive, and iCloud where available, with local files you can inspect.
