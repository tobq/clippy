# Security

BoardClip is a local-first clipboard history app. Clipboard contents can be sensitive, so please report security issues privately.

## Reporting

Email the maintainer or open a private GitHub security advisory if available. Please include:

- A short description of the issue.
- Reproduction steps.
- Affected platform and version.
- Any relevant logs or screenshots.

## Notes

- BoardClip stores clipboard history locally in the app directory.
- Native cloud sync uses folders already configured on the machine, such as Google Drive, OneDrive, or iCloud.
- The public install scripts are plain text and should be inspected before running if you are unsure.
